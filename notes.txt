COMMANDS
- RUN
- CONFIG
- INSPECT

** RUN **
setup midi virtual out
launch n control
start sending midi

** CONFIG **
ask user for key binds (vol up, vol down, mute)
ask user for channels to send to (can be a list) // maybe separate by whitespace
ask for retrigger speed - per command option to no retrigger


** INSPECT ** 
list keybinds + throttling speeds
list channels


** FLOW **
key binds for (vol up, vol down, mute)
CGEvent sends found key bind

On Mount:
read config.txt
save (key, flag) -> message
setup midi
read CGEvent -> (key, flag)
check if (key, flag) should retrigger
if so, send message
